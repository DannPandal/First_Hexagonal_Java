# Ejemplo Hexagonal en Java

Este es un ejemplo de una aplicación Java con arquitectura hexagonal.


## Ejecución

Para ejecutar la aplicación, se debe ejecutar el siguiente comando:

```bash
mvn spring-boot:run
```
